# Broken Arrow Outdoors

Next.js App Router storefront using Shopify Storefront API.

## Run locally

1. Install dependencies:
   `npm install`
2. Copy `.env.example` to `.env.local` and fill in your Shopify token.
3. Start development:
   `npm run dev`

## Required environment variables

- `SHOPIFY_DOMAIN` (example: `6c6a1d-39.myshopify.com`)
- `SHOPIFY_API_VERSION` (example: `2025-01`)
- `STOREFRONT_PUBLIC_TOKEN` (Shopify Storefront API public token)
